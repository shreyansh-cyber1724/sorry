# Sorry Website 💜

A small interactive apology website made with warmth, emotions, animations, music, and a little bit of effort.

This project was created to make someone smile again after a misunderstanding between close friends.

Visit - https://hellscholar.github.io/SORRY-WEBSITE/

## Features

- Smooth animated UI
- Emotional story sections
- Fake chat conversation
- Interactive buttons
- Background music
- Hidden surprise message
- Mobile-friendly design

## Message Behind It

Sometimes a simple “sorry” doesn’t feel enough.  
So this website was made to say:

> “You matter to me, and I don’t want things to stay weird between us.”

## Note

This is not a romantic project.  
It’s simply a heartfelt apology website for a close friend.
